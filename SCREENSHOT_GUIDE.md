# Screenshot Guide — StoryPal App Store Listing

## Required Sizes
- iPhone 6.7" (1290 x 2796) — iPhone 15 Pro Max
- iPhone 6.5" (1242 x 2688) — iPhone 11 Pro Max
- iPad 12.9" (2048 x 2732) — iPad Pro
- Android Phone (1080 x 1920)
- Android Tablet (1200 x 1920)

## Design Guidelines
- Background: Use app's cream background (#FFF8F0) or extend with subtle gradient
- Text overlay: Use bold white text on gradient strip at top/bottom
- Font: System SF Pro (iOS) or Roboto (Android) — bold weight
- Badge overlays: "NEW" or "AI-Powered" badges in coral (#FF6B6B)
- Do NOT crop the floating tab bar — it's a visual differentiator
- Ensure status bar shows realistic time (9:41 AM for iOS)

## App Icon Spec
- Size: 1024x1024 (no transparency, no rounded corners — Apple adds them)
- Design: Open storybook with sparkles/stars, warm coral-to-orange gradient background
- Style: Flat/modern with subtle depth, child-friendly
- Colors: Primary gradient (#FF6B6B to #FF8E53) background, white book icon with sparkles

## Feature Graphic (Android)
- Size: 1024 x 500
- Design: App icon + "StoryPal" text + tagline "AI-Powered Stories for Kids"
- Background: Primary gradient with subtle star/sparkle pattern
